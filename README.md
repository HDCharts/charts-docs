## Repo Layout

- `docs-app/` - Next.js docs website
- `content/` - versioned markdown content
- `registry/versions.json` - version registry used by the site
- static assets for API/demo/playground are served from object storage/CDN

## Local Development

```bash
cd docs-app
npm run dev
```
